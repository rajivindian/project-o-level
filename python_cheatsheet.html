Basics

Print Statement

print("Hello, World!") print("Name:", name) print(a, b, sep="-")

Comments

# Single line comment """ Multi-line comment using triple quotes """

Input

name = input("Enter name: ") age = int(input("Age: "))

Variable Assignment

x = 5 y = x a, b = 1, 2 x = y = z = 0

Data Types

String

s = "Hello" s = 'Single quotes' s = """Multi line"""

Numbers

x = 10 # int y = 3.14 # float z = 2j # complex

Boolean

flag = True flag = False result = True and False

Type Conversion

int("5") # 5 str(123) # '123' float("3.14") # 3.14 bool(1) # True
Type Checking: Use type(variable) to check the data type of a variable.

Operators

Operator Description Example
+, -, *, / Addition, Subtraction, Multiplication, Division 5 + 3 = 8
// Floor Division 7 // 2 = 3
% Modulus (Remainder) 7 % 2 = 1
** Exponentiation 2 ** 3 = 8
==, !=, <, >, <=, >= Comparison Operators 5 == 5, 3 < 5
and, or, not Logical Operators x > 0 and x < 10
&, |, ^ Bitwise Operators 5 & 3, 5 | 3

Control Flow

If Statement

if x > 5: print("Greater") elif x == 5: print("Equal") else: print("Less")

For Loop

for i in range(5): print(i) for item in list: print(item)

While Loop

while x < 10: print(x) x += 1 while True: if condition: break

Loop Control

break # Exit loop continue # Skip iteration pass # Do nothing
Range Function:
range(5) # 0, 1, 2, 3, 4 range(1, 5) # 1, 2, 3, 4 range(0, 10, 2) # 0, 2, 4, 6, 8

Functions

Define Function

def greet(name): return "Hello, " + name result = greet("Alice")

Default Parameters

def power(x, n=2): return x ** n power(5) # 25 power(5, 3) # 125

Variable Arguments

def sum_all(*args): return sum(args) sum_all(1, 2, 3, 4)

Keyword Arguments

def info(**kwargs): for key, value in kwargs.items(): print(key, value) info(name="Alice", age=25)

Lists & Tuples

List Creation

lst = [1, 2, 3, 4] lst = [1, "hello", 3.14] lst = list() lst = [x for x in range(5)]

List Methods

lst.append(5) lst.insert(0, 0) lst.remove(3) lst.pop() lst.sort() lst.reverse()

List Indexing

lst[0] # First element lst[-1] # Last element lst[1:3] # Slice (1 to 2) lst[::2] # Every 2nd element

Tuples

tup = (1, 2, 3) tup = tuple() tup[0] # Access # Immutable - cannot modify

Strings

String Methods

s.upper() # HELLO s.lower() # hello s.strip() # Remove spaces s.replace("a", "b") s.split() # Split by space s.find("sub") # Find index

String Formatting

name = "Alice" f"Hello {name}" "Hello {}".format(name) "Hello %s" % name

String Indexing

s = "Python" s[0] # 'P' s[1:4] # 'yth' s[-1] # 'n' len(s) # 6

Dictionaries

Dictionary Creation

d = {"name": "Alice", "age": 25} d = dict() d["city"] = "NYC"

Dictionary Methods

d.keys() # All keys d.values() # All values d.items() # Key-value pairs d.get("name") # Get value d.pop("age") # Remove & return

Dictionary Access

d["name"] # Access value d.get("age", 0) # Default value "name" in d # Check key

File Handling

Reading Files

f = open("file.txt", "r") content = f.read() lines = f.readlines() f.close()

Writing Files

f = open("file.txt", "w") f.write("Hello") f.close() # Append mode f = open("file.txt", "a")

With Statement

with open("file.txt", "r") as f: content = f.read() # File auto-closes
File Modes: "r" (read), "w" (write), "a" (append), "x" (create), "b" (binary)

Exception Handling

Try-Except

try: x = 1 / 0 except ZeroDivisionError: print("Error!")

Multiple Exceptions

try: x = int("abc") except (ValueError, TypeError): print("Invalid input")

Finally Block

try: f = open("file.txt") finally: f.close()

Raise Exception

if x < 0: raise ValueError("Must be positive")
Common Exceptions: ValueError, TypeError, KeyError, IndexError, ZeroDivisionError, FileNotFoundError

Built-in Functions

Function Description Example
len() Length of object len([1, 2, 3]) = 3
sum() Sum of elements sum([1, 2, 3]) = 6
max(), min() Maximum/Minimum value max([1, 2, 3]) = 3
sorted() Return sorted list sorted([3, 1, 2]) = [1, 2, 3]
enumerate() Get index and value for i, v in enumerate([1, 2])
zip() Combine iterables zip([1, 2], [3, 4])
map(), filter() Transform/filter items map(int, ['1', '2'])
any(), all() Check conditions any([False, True]) = True
abs(), round() Absolute value, Rounding abs(-5) = 5

Tips & Tricks

List Comprehension

squares = [x**2 for x in range(5)] evens = [x for x in lst if x % 2 == 0]

Lambda Functions

square = lambda x: x**2 square(5) # 25 sorted(lst, key=lambda x: x[1])

String Slicing

s = "Python" s[::-1] # "nohtyP" (reverse) s[1:4] # "yth" s[::2] # "Pto"

Unpacking

a, b = (1, 2) x, *middle, y = [1, 2, 3, 4, 5] # x=1, middle=[2,3,4], y=5
