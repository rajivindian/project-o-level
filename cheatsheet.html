1. Python Basics

How to print and use variables.

name = "Rajiv"
age = 23
print(f"My name is {name} and I am {age}")

2. Loops

Repeating tasks with 'For' loops.

for i in range(5):
    print(i)  # Prints 0, 1, 2, 3, 4

3. Tkinter (GUI)

Creating a basic window.

import tkinter as tk
root = tk.Tk()
root.title("My App")
root.geometry("300x300")
root.mainloop()

4. File Handling

Moving files using Shutil (Important for Exam!).

import shutil
# Move file from Source to Destination
shutil.move("source.txt", "destination_folder/")
